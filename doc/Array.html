- (Hash) extract_options!

Extract options from a set of arguments. Removes and returns the last element in the array if it's a hash, otherwise it returns a blank hash.

def options(*args)
  args.extract_options!
end

options(1, 2)           # => {}
options(1, 2, :a => :b) # => {:a => :b}

Returns:

  • (Hash) —

    a hash of options extracted from the array
